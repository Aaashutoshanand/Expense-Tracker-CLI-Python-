# 💸 Expense Tracker CLI (Python)

A simple command-line expense tracker built using Python.
This project allows users to add, view, and calculate expenses using a CSV file.

---

## 🚀 Features

* ➕ Add expenses (description + amount)
* 📊 View all expenses
* 💰 Calculate total spending
* 📁 Stores data in CSV file
* 🧠 Modular Python functions

---

## 🛠 Tech Stack

* Python 3
* CSV Module
* File Handling
